# Go Sniff

*Lightweight packet sniffing utilities written in pure Go.*

No CGO. No libpcap dependency headaches. Just pure Go reading packets off the wire with the quiet confidence of a well-placed `tcpdump`.
We keep it minimal so you can keep it deployed.
